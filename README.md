Modern landing page of a SaaS tool — built for clean UX, responsive design, and production-ready SEO.

---

## 💼 Skills & Tech Used

- **Framework:** Next.js
- **UI:** React, React DOM
- **SEO:** `next-sitemap`
- **Data Fetching:** TanStack React Query
- **Forms & Validation:** React Hook Form, `@hookform/resolvers`, Zod
- **UI Components:** Radix UI (Accordion, Dialog, Select, Tabs, Tooltip, etc.)
- **Theming:** `next-themes`
- **Animations:** Framer Motion + Motion
- **Date UI:** React Datepicker, React Day Picker, date-fns
- **UI/UX:** Sonner, Vaul, cmdk, embla-carousel-react, input-otp, react-resizable-panels
- **Icons:** Lucide React
- **Styling Utils:** `clsx`, `class-variance-authority`, `tailwind-merge`
- **Code Quality:** ESLint, `eslint-config-next`
- **CSS Tooling:** Tailwind CSS (v4), PostCSS, Autoprefixer, `@tailwindcss/postcss`, `tw-animate-css`

---

## 🚀 Run Locally

```bash
git clone https://github.com/muhamadzainalam-dev/LandingPageApp.git
cd LandingPageApp
npm install
npm run dev
```

Open:
```text
http://localhost:3000
```
